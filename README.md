# Automated File Organizer

A Python script that automatically organizes files into folders based on file type.

## Features
- Organizes files into categories (Images, Documents, Videos, etc.)
- Prevents overwriting existing files
- Safe testing with a dedicated folder

## Usage
1. Create a folder: ~/Desktop/automation
2. Add files to the folder
3. Run:

python3 organizer.py
